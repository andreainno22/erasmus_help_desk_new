\chapter{Deployment}

\section{Sviluppo}
Esecuzione locale:
\begin{verbatim}
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
\end{verbatim}

\section{Porta 8000}
Per liberare la porta su Windows (PowerShell):
\begin{verbatim}
Get-NetTCPConnection -LocalPort 8000 -State Listen |
  Select -ExpandProperty OwningProcess -Unique |
  % { Stop-Process -Id $_ -Force }
\end{verbatim}
